## 빌드하고 실행하기
1. 콘솔로 이동하기 (저는 git bash를 사용했습니다.)
2. cd '프로젝트 폴더 위치'
2. ./gradlew build
3. cd build/libs
4. java -jar hello-spring-0.0.1-SNAPSHOT.jar
5. 실행확인
